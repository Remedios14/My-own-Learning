函数的两种定义方式：
        1. 使用 function 关键字来定义，定义时传入参数无需指明类型，如 python
            function 函数名(){
                函数体;
                [return val;]
            }
        2. 基本没有区别
            var 函数名 = function(形参列表){
                函数体;
            }
    JS 不允许函数重载，但不报错，默认用更近的同名函数来执行，不论传入参数问题（也和 python 一样）

    注意每个函数体自带一个 arguments 属性(隐形参数)，属于一个数组，其内元素为该函数的形参
    可以在函数体内调用该数组执行遍历等操作

    不能重载就是这个隐形参数搞的鬼
