    事件是电脑输入设备与页面进行交互的响应行为。

常用的事件：
    onload 加载完成事件：       发生于页面加载完成之后，常用与做页面 JS 初始化操作
    onclick 单击事件：         常用于按钮的点击响应操作
    onblur 失去焦点事件：       常用于输入框失去焦点后验证其输入内容是否合法
    onchange 内容发生改变时间：  常用于下拉列表和输入框内容发生改变后操作
    onsubmit 表单提交事件：     常用于表单提交前，验证所有表单项是否合法。

事件的注册又分为静态注册和动态注册两种：
   注册：即让浏览器知道当事件响应后要执行哪些操作代码的行为，称为事件注册或事件绑定

静态注册事件： 通过 html 标签的事件属性之间赋予事件响应后的代码，这种方式我们叫静态注册

动态注册事件： 指先通过 js 代码得到标签的 dom 对象，然后再通过 dom 对象.事件名 = function(){} 这种
形式赋予事件响应后的代码，叫做动态注册。

    动态注册基本步骤：
        1、获取标签对象
        2、标签对象.事件名 = function(){}
