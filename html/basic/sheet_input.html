关于提交内容的说明:
    表单提交的时候，数据没有正确地发送给服务器的三种情况：
        1、表单项没有 name 属性 —— 加上后传送数据为 filled_name=input_val
        2、单选、复选（下拉列表中的option标签）都需要添加 value 属性，以便在被选中时发送给服务器
        3、表单项不在提交的 form 标签内部

    GET 请求的特点是：
        1、转向页面上，浏览器地址栏中的地址是：action 属性的值[+?+请求参数]
            请求参数的格式是： name=input_val&name=input_val
        2、不安全，会在地址中显示
        3、有数据长度的限制（好像是100个字符，反正不用这个就对了）
    
    POST 请求的特点：
        1、浏览器地址栏中只有 action 属性值
        2、相对于 GET 请求安全
        3、理论上没有数据长度的限制
