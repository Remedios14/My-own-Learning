一些旧的浏览器不支持 iframe。

如果浏览器不支持 iframes 则不会显示。

iframe 内部属性表

属性 含义
src 填入URL链接，可以是本地的html文件
width/height 设置宽高，默认以像素为单位，也可改成按比例
frameborder 设置边框宽度，默认0无边框
name 命名后可以在链接中指定 target 使得在子界面转到 url 

    有一些网站不希望被嵌套，其头文件内有一选项
        X-Frame-Options
    他有三个可配置的值
    

        DENY：表示该网站页面不允许被嵌套，即使是在自己的域名的页面中也不能进行嵌套
        SAMEORIGIN：表示该页面可以在相同域名页面中被嵌套展示。
        ALLOW-FROM url：表示该页面可以在指定来源页面中进行嵌套展示。
